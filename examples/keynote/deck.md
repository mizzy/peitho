# コンテンツとデザインを分ける

内容はMarkdownに、見た目はテンプレートとCSSに。書いている間、装飾のことは考えない。

# テンプレートはスキーマ

スロットの過不足も、型の不整合も、参照切れも、ビルドが行番号付きで止める。黙って落ちるものは何もない。
